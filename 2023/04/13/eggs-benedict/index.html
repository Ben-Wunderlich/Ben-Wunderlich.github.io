eggs-benedict
2023-04-13 #Kotlin #Android

I didn’t realize I loved eggs benedict until I had begun comparing comparing each benny I ate to all the previous ones I had enjoyed. Someone suggested I track it, so I made a quick android app to do this, I took the opportunity to get used to using sql database integration with android. As it turns out, it is incredibly easy to use, so I also added a gui that lets you order records by each column of the database.

I also learned from my mistakes in previous projects and added a download button, so I can still get the database even if I lose all the app development files or something else goes wrong.

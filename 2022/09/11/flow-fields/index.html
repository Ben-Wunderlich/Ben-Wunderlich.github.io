flow-fields
2022-09-11 #Art #Javascript

Flow fields are a common element in generative art that I hadn’t explored yet because I had seen so many other people explore this topic.

How flow fields work

Fields

The field in ‘flow field’ refers to a field of angles that each point in some direction.

Flows

Once the field is complete, you can start somewhere on the field and move in the direction the field points, do this repeatedly and you can trace out the path of that flow.

Making fields interesting

How the field angles are set is one of the main variables that can change the overall look of the image. In the demo images above I used perlin noise, commonly used for flow fields.

Here I made it so each point on the field checks the distance to 6 points and uses the smallest distance as the angle.

And here I set the angle to be sin of its x position + y position.

Changing flow rendering

Along with the field angles you can also adjust how each flow traces its path. For the last few I made the line segments smaller so it looks round and had the flows start dark then get lighter as they went.

For this one I started each flow with black but had it morph with each jump, then average it with a specific color. For its field I used sin waves combined with circular motion as the angle at each point.

The next image is similar to the last, just using a different color. It uses the best function(xsinx) with distance to center, combined with circular motion.

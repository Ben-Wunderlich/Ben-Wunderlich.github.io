spiral-automata
2022-11-28 #Art #Javascript

basics

I made a p5js framework for cellular automata, it treats each cell as an independent object that determines what to do based on the number of adjacent cells.

This first one spirals outwards if there are more than 3 cells nearby, otherwise it spirals inwards.

start circling

I wanted to make an instruction set that produced more interesting patterns. This one goes straight out if there are more than 4, goes to the middle if it has no neighbors, and rotates around center if it is between 1 and 4.

click on the buttons to control the automata.

four stars

This one rotates 4 ‘stars’ in a pattern, and each cell rotates around the closest star, it was inspired by binary star systems.

accidents can be better

After all the rotation, I wanted cells that go around in a square, so the system below moves away from the center if there are more that 4 neighbors, spirals inwards if there are no neighbors and goes in a square if there are between 1 and 4 neighbors.

I made a mistake when I was creating the code so if cells are within a center square they are immobile. However I liked the look of this and so kept this feature.

“it’s not a bug, it’s a feature”

wavy wavy

I wanted to use the best function x*sin(x), so I created this unusual system, cells move towards the center, once a certain amount of cells touch the center it switches and they all go outward till one touches the edge. The only individual decisions work to push the cells apart so they don’t clump up too much.

click on the buttons to control the automata.

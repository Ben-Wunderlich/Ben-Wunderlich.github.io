python-tools
2022-07-03

There are a lot of small problems out there, so over the years I have made a lot of small python scripts to make my life easier.

file organizer

I like to keep photos from my phone on my computer, I used the microsoft photos app to do this, but it steadily started working worse. First it just started putting them all into one folder instead of organizing them, then it stopped working at all. I could still manually go folder by folder and bring them in, so I made a python script to copy new images and organize them into folders based on date.

webp deleter

I hate when I download an image and it is in the .webp format, you can use websites to convert them or open them each with a photo editor and save as a jpg, but those are both a lot of work. To save me the effort of file conversion I made a “bomb script” that consumes every webp file it can find in the current directory and converts it to the jpg format.

pdf combiner

During the pandemic of 2020 we had to start submitting assignments online, this meant I had to do a lot of taking pictures of paper assignments and submitting them as pdfs. For assignments this was fine, but submitting exams at the last minute using a website was nerveracking and too slow, so I made a simple python script to rapidly combine images into pdfs.

gif maker

I got tired of using many different tools to make gifs from images I made, so I just made a quick script to do it.

signature maker

My sister wanted something that I made for her wall, so I figured I should make a signature to put on them. I talked to my artistic sister who helped me make a signature, then I made another “bomb script” that applied the signature to every image in the folder, scaling it up to an appropriate size regardless of image size.

static

I kept needing images of pure static for various projects, so I made a script to make any number of static images of any size. I can’t remember using it since I made it, I just find it amusing that I took the time to make it.

pixel rotation

At some point I rotated each pixel value in an image so the new ‘rgb’ values are the original ‘brg’, it was slow so I made a dedicated bomb script to find out which images looked good rotated. This was neat but had an unexpected side effect, the jpg images I did this to started to look crispy, the repeated saving slowly created glitchy artifacts.
